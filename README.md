# S-ries_Temporais_ME607
Este repositório tem como objetivo compartilhar materiais feitos na disciplina de Séries Temporais na minha graduação

No trabalho 1, tive como objetivo escolher um modelo não apresentado em aula, explicar e ilustrar o modelo em dados. Portanto escolhi o modelo Prophet da META, e ilustrei ele em dados da produção industrial do Brasil dos últimos anos. Dessa forma fiz um documento tex para fazer o trabalho e utilizei algumas referências apresentadas em aula, além do modelo prophet, usei o python para aplicar o modelo nos dados escolhidos.
